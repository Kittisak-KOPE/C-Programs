## Pointers in C programming

### Pointer

```
int xyz = 10;
int *pt;

pt = &xyz;

printf("The address of variable xyz is %x \n", &xyz);

printf("The address of variable xyz is %x \n", pt);

printf("The address of variable pt is %d \n", *pt);
```

### Null Pointer

```
int *pt = NULL;
int xyz = 4;

pt = &xyz;

printf("The address of *pt is %x \n", pt);
```
